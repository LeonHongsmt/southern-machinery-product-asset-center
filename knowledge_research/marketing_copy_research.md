# Southern Machinery SMT / THT 设备英文落地页文案研究

> 本文只做英文营销文案结构研究，用于后续把 Southern Machinery 的 SMT / THT 设备资料、PDF、图片、说明书转化为英文落地页内容。  
> 研究来源包括 Southern Machinery / SMThelp 官网页面、产品页、FAQ、证书页、产品 PDF 与公开产品目录信息。

## 1. 英文 headline 写法

Southern Machinery 当前页面常见 headline 有三类：

- 品类解决方案型：`Boost Your Manufacturing Efficiency with Southern Machinery's SMT THT Machine Solutions`
- 产品型号型：`S7020 Series PIN Eyelet Terminal Insertion Machine`
- 技术/产线型：`THT Axial Radial Auto Insertion Machine with Loader Unloader Line for Smart EMS Factory PCB Assembly`

更适合英文落地页的 headline 应该同时包含：

- 核心设备名：如 `Radial Insertion Machine`、`Wave Soldering Machine`、`Odd Form Insertion Machine`
- 使用场景：如 `PCB Assembly`、`THT Production`、`Smart EMS Factory`
- 主要价值：如 `higher throughput`、`reduced labor cost`、`consistent insertion quality`
- 如有可靠来源，可加入硬指标：如 `12,000 CPH`、`18,000 CPH`、`22,000 CPH`

推荐模板：

```text
Automate [SMT/THT Process] with a Reliable [Machine Type] for PCB Assembly
```

```text
[Model Name] [Machine Type] for High-Efficiency [Application / Component Type] Insertion
```

```text
Improve Throughput and Consistency in [THT / SMT / DIP] Production
```

具体示例：

- `S7020 Dual-Head PIN Eyelet Terminal Insertion Machine for High-Speed PCB Assembly`
- `S-3010B Radial Insertion Machine for Cost-Efficient THT Automation`
- `Lead-Free Dual Wave Soldering Machine for Stable High-Volume PCB Production`
- `Odd Form Insertion Automation for Smarter DIP Line Manufacturing`

注意：headline 可以强化价值，但不能虚构“全球第一”“行业最低成本”“100%无缺陷”等没有证据的绝对化表达。

## 2. 英文 subheadline 写法

Southern Machinery 现有 subheadline 高频表达是：

- `Specialized in Design and Manufacturing SMT THT machine for Smart EMS Factory`
- `Southern Machinery provides professional SMT peripheral equipment`
- `We mainly design and manufacture SMT/AI machine, also providing full services support and selling spare parts for electronic manufacturing PCB assembly`

落地页 subheadline 应该比 headline 更具体，回答 3 个问题：

- 这台设备做什么？
- 解决哪类生产问题？
- 为什么适合采购询盘？

推荐结构：

```text
Designed for [process / component type], this [machine] helps electronics manufacturers [benefit 1], [benefit 2], and [benefit 3] with [source-backed feature].
```

示例：

```text
Designed for THT PCB assembly, the S-3010B automates radial component insertion to reduce manual plug-in work, improve consistency, and support higher output per floor space.
```

```text
Built for PIN, eyelet, and terminal insertion, the S7020 combines dual-head insertion, visual programming, and flexible feeding options for efficient, repeatable PCB production.
```

```text
Southern Machinery provides SMT and THT automation equipment, spare parts, repair support, upgrade services, and training for electronics manufacturing and PCB assembly lines.
```

文案语气建议：

- 使用 `designed for`、`built for`、`engineered for`，更像 B2B 工业设备文案。
- 避免中文式堆词，例如 `high quality good price machine`。
- 避免空泛句，例如 `best machine for everyone`。

## 3. Key benefits 写法

Key benefits 适合放在首屏下方或产品概览区，用 4-6 条表达采购关心的业务价值。Southern Machinery 资料里可提炼的利益点包括：

- 减少人工插装和培训成本
- 提高产能和产线稳定性
- 改善插装质量、一致性和可追溯性
- 适配多种元件、PCB 尺寸、供料方式
- 节省生产空间
- 支持维护、培训、备件、升级服务

推荐格式：

```text
- Reduce Manual Insertion Labor
  Automate repetitive THT insertion work and reduce dependence on operator skill level.

- Improve Output Consistency
  Support stable insertion quality and reduce common DIP defects such as missing, wrong, or lifted components.

- Support Flexible Production
  Handle different PCB sizes, insertion angles, component packages, or feeding methods when supported by the machine specification.

- Lower Total Operating Pressure
  Reduce rework, material waste, manual handling, and floor-space pressure in PCB assembly production.
```

按产品类型可写：

### THT auto insertion

- `Reduce manual DIP insertion work`
- `Improve insertion consistency and process repeatability`
- `Support axial, radial, jumper wire, odd-form, PIN, eyelet, or terminal components`
- `Increase throughput with model-specific CPH data`

### SMT peripheral equipment

- `Improve process preparation and line stability`
- `Support cleaning, handling, inspection, storage, and material management`
- `Reduce downtime caused by contamination, manual handling, or material errors`

### Wave soldering

- `Support lead-free and common solder production`
- `Improve conveyor stability and soldering process control`
- `Use specification-backed temperature, conveyor, preheating, and control-system details`

## 4. Applications 写法

Applications 不应只写行业名，要写“设备用于什么 PCB / 元件 / 生产场景”。Southern Machinery 资料中可支撑的应用方向包括：

- EMS PCB assembly provider
- Household appliances / white goods PCB assembly
- Washing machine control PCB
- LED traffic signal PCB
- Tesla / electric car controller PCB 示例视频标题
- THT / DIP production line
- Smart EMS factory PCB assembly
- High-volume mass production
- Small-batch or delicate manufacturing
- Prototype / R&D / low-to-medium volume SMT assembly

推荐格式：

```text
Ideal Applications

- EMS PCB Assembly Lines
  For factories that need automated SMT/THT equipment, peripheral machines, and process support.

- THT / DIP Production
  For replacing repetitive manual insertion work with axial, radial, odd-form, PIN, eyelet, or terminal insertion automation.

- Home Appliance Control Boards
  Suitable when the source material confirms application in TV, washing machine, or white goods PCB assembly.

- LED and Power Electronics Boards
  Use only when the product page or project material confirms compatibility with those components or board types.
```

Applications 写法原则：

- 可以从客户痛点推导场景，但具体行业案例必须来自资料。
- `automotive`、`medical`、`aerospace` 等高信任行业不能随便写，除非资料明确出现。
- 如果只知道“PCB assembly”，不要扩写成“all electronics industries”。

## 5. Technical advantages 写法

Technical advantages 应该从参数、结构、控制系统、软件、检测、服务能力中提炼。它和 key benefits 的区别是：benefits 写客户结果，technical advantages 写设备为什么能带来结果。

Southern Machinery 页面中可提炼的技术点：

- S7020：双插入头、12,000 CPH、bulk / reel terminal feeding、360 度插入角度、visual programming、Excel、USB import、0.001 mm / pulse motion accuracy。
- S-3010B：四种 span 2.5 / 5.0 / 7.5 / 10.0 mm、22,000 CPH 标称速度、13,000 CPH 实际速度、0-360 度插入方向、machine vision correction、English / Chinese interface。
- S4000：18,000 CPH、10-100 inputs expandable sequencer、component replenishment without machine stoppage、manual or automatic PCB load/unload、Windows-based interface。
- S-WS350B：PC + PLC 控制、300-1800 mm/min conveyor speed、3-zone preheating、PID + SSR temperature control、Siemens PLC、lead-free / common solder support。
- S-70LD：适用于替代人工 odd-form / DIP 插装，解决人工不稳定、误插、漏插、浮高、产能低等问题。
- 证书与服务：CE、ROHS、ISO14001、ISO9001、专利、培训、操作手册、视频、备件、维修、升级。

推荐写法：

```text
Technical Advantages

- Vision-Assisted Position Correction
  Helps improve insertion accuracy and reduce manual alignment dependency.

- Flexible Component Handling
  Supports [component type] with [feeding method / span / angle] based on the machine specification.

- Production-Oriented Control System
  Uses [PLC / PC / touch screen / Windows interface] for easier operation, diagnostics, and recipe management.

- Traceable Quality Control
  When supported by the machine, use sensing, data storage, or inspection functions to support process traceability.
```

重要：技术优势必须尽量绑定具体型号。不要把 S7020 的 `12,000 CPH` 写到 S-70LD，也不要把 S-WS350B 的 `Siemens PLC` 泛化成所有设备都有 Siemens PLC。

## 6. FAQ 写法

Southern Machinery FAQ 当前覆盖：

- 是否容易操作
- 机器故障后怎么办
- MOQ
- 如何购买
- 如何付款
- 质量保证和出货检查
- 包装方式
- 交期
- 收到货不能工作怎么办
- 证书
- 是否有英文版本
- 软件支持
- 培训方式

英文落地页 FAQ 应该更采购导向，建议分成“购买前”“技术确认”“售后服务”三组。

推荐 FAQ 模板：

```text
Q: Is the machine easy to operate for first-time users?
A: Yes. Southern Machinery provides English manuals, guide videos, and technical support. For model-specific operation steps, refer to the user manual and training materials.

Q: Can the machine be customized for my PCB or component?
A: Some machines support customized PCB size, feeding method, insertion span, or production configuration. Please share your PCB drawing, component details, and production requirement for confirmation.

Q: What information should I provide before requesting a quote?
A: Please provide the product model, PCB size, component type, expected capacity, line layout, voltage requirement, and any automation requirement such as loader, unloader, inspection, or MES connection.

Q: What happens if the machine has a problem after delivery?
A: Contact the sales and technical team with photos, videos, alarm messages, and a description of the issue. Warranty, spare parts, and repair support should follow the confirmed quotation or contract terms.

Q: Do the machines have certificates?
A: Southern Machinery states that its machines have passed CE and ROHS certification, and its certificate page also mentions ISO14001 and ISO9001. Use certificate claims only when the matching certificate is available for the specific model.

Q: Do you provide installation and training?
A: Southern Machinery materials mention English manuals, guide videos, training, and worldwide support. For each order, confirm whether on-site installation, remote training, or video support is included.
```

FAQ 写法注意：

- 不要承诺固定交期、固定保修期、免费安装、免费备件，除非对应报价或页面明确写了。
- 支付方式、包装、运费、售后责任要保留 `subject to confirmed quotation / invoice / contract`。
- 对复杂设备，FAQ 要引导客户提供资料，而不是直接承诺兼容。

## 7. CTA 文案

Southern Machinery 当前 CTA 常见：

- `REQUEST A QUOTE`
- `Get a quote`
- `Call to discuss SMT machine`
- `Email to SMT Specialist`
- `More pictures please click SMT THT Albums`
- `Looking for 3D machine model`
- `Looking for SMT machine Catalog`
- `pdf download`

英文落地页 CTA 应区分不同意图：

### 主 CTA

- `Request a Quote`
- `Get Machine Recommendation`
- `Talk to an SMT/THT Specialist`
- `Send Your PCB & Component Requirements`

### 次 CTA

- `Download Catalog`
- `View Machine Photos`
- `Watch Working Video`
- `See 3D Machine Model`
- `Ask for Technical Specification`

### 表单按钮

- `Submit Inquiry`
- `Send Requirement`
- `Request Technical Confirmation`

### CTA 附近辅助文案

```text
Share your PCB size, component type, expected capacity, and automation requirements. Our team will help confirm the suitable SMT/THT machine configuration.
```

```text
Not sure which model fits your production line? Send us your board and component details for a recommendation.
```

避免：

- `Buy Now`：工业设备采购流程通常不是即时购买。
- `Best Price Now`：容易显得低端，也可能引发价格承诺风险。
- `Guaranteed ROI`：除非有项目测算和合同依据。

## 8. 如何把 PDF、图片、说明文档内容转成英文营销文案

转化流程建议：

### 第一步：提取事实

从资料中提取以下字段：

- 产品型号：如 `S7020`、`S-3010B`、`S-WS350B`
- 设备类型：如 `PIN Eyelet Terminal Insertion Machine`
- 适用工艺：SMT / THT / DIP / wave soldering / cleaning / board handling
- 核心参数：速度、PCB 尺寸、电源、重量、尺寸、精度、插入角度、温控范围
- 控制系统：PLC、PC、touch screen、Windows interface、software support
- 供料方式：bulk、reel、vibration bowl、feeder
- 检测与质量：vision correction、leakage detection、pin penetration sensing、traceability
- 服务资料：manual、video、training、spare parts、repair、upgrade、warranty

### 第二步：把参数翻译成客户价值

示例：

- 原始参数：`Speed: 12000 CPH`
- 营销表达：`Supports high-speed terminal insertion up to 12,000 CPH for efficient PCB assembly.`

- 原始参数：`Insertion angle: 360° increment 1°`
- 营销表达：`Allows flexible insertion orientation with 360-degree angle control in 1-degree increments.`

- 原始参数：`Programming: online visual programming, visual correction, Excel`
- 营销表达：`Simplifies production setup with visual programming, correction, and Excel-based data preparation.`

- 原始痛点：人工不稳定、漏插、误插、浮高
- 营销表达：`Helps reduce common manual DIP insertion issues such as missing, wrong, or lifted components.`

### 第三步：生成页面模块

建议页面结构：

1. Hero：headline、subheadline、主 CTA、关键参数 3-4 个
2. Problem：人工、质量、产能、空间、培训成本等痛点
3. Solution：该机器如何解决
4. Key Benefits：业务价值
5. Technical Advantages：参数和技术依据
6. Applications：适用 PCB、元件、工艺、行业
7. Specification Table：保留原始参数，不营销化改写
8. Media：图片、视频、3D model、PDF catalog
9. Service & Support：培训、手册、备件、维修、升级
10. FAQ：购买、技术、售后问题
11. CTA：询盘表单

### 第四步：图片转文案

图片可转化为：

- 设备外观：`compact floor-standing design`、`dual-head structure`、`integrated conveyor`
- 局部特写：`reinforced titanium claw`、`touch screen control panel`、`vibration bowl feeder`
- 产线照片：`inline production configuration`、`loader/unloader integration`
- 证书照片：`CE / ROHS certificate available for model confirmation`

但图片不能单独证明：

- 具体产能
- 精度数值
- 认证范围
- 客户行业
- 售后承诺

### 第五步：说明书转文案

说明书适合生成：

- 操作便利性：根据界面、步骤、软件功能提炼
- 维护便利性：根据保养、故障排查、备件章节提炼
- 安全性：根据安全注意事项和报警机制提炼
- 培训材料：说明有 `user manual` / `operation manual` / `guide video`

不要把说明书里的所有操作步骤放到落地页，应提炼为采购能理解的价值。

## 9. 文案中不能乱编、必须来自资料的内容

以下内容必须来自 PDF、产品页、证书、说明书、报价单、测试视频或客户确认资料：

- 型号名称、设备名称、产品系列归属
- 速度 / CPH / 产能 / 节拍
- PCB 尺寸、元件尺寸、lead span、插入角度、精度、重量、外形尺寸
- 电源、电压、气压、锡炉温度、预热区、输送速度、温控精度
- 控制系统品牌：如 Siemens、Delta、OMRON、MEANWELL、SMC 等
- 是否支持 MES / ERP / traceability / USB import / Excel / visual programming
- 是否支持定制，以及定制范围
- 证书：CE、ROHS、ISO14001、ISO9001、专利
- 保修期、免费备件、安装培训、响应时间、交期、付款方式
- 客户案例、国家地区、行业应用、具体产线案例
- ROI、节省比例、人工替代人数、成本下降比例
- “全球”“第一”“领先”“最优”“100%”等强声明

可以合理改写但不能改变事实的内容：

- `easy to operate` 可以写成 `designed for easier operation with English manuals and guide videos`
- `reduce labor costs` 可以写成 `helps reduce manual labor pressure`
- `improves product quality and consistency` 可以写成 `supports more consistent insertion quality`
- `can be customized` 可以写成 `custom configuration may be available after technical confirmation`

建议使用保守表达：

- `helps reduce`
- `supports`
- `designed to improve`
- `can be configured for`
- `based on confirmed machine specification`
- `subject to final technical confirmation`
